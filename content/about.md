## About WebTool

WebTool.app is a collection of utility tools built to solve everyday problems.

### Current Tools

| Tool | Description |
|------|-------------|
| [OCR](/ocr) | Convert documents to structured Markdown |

### Philosophy
- **Simple:** No accounts, no complexity
- **Private:** Data is not stored long-term
- **Free:** Tools are available to use at no cost
