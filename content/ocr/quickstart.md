## Access the Tool

Visit [ocr.webtool.app](https://ocr.webtool.app) to start using the OCR tool.

## Process Your First Document

### 1. Upload

Drag and drop files onto the upload area, or click to browse.
**Supported formats:** PDF, PPTX, DOCX, PNG, JPG, GIF, WebP
**File size limit:** 50MB per file

### 2. Process

Click the upload button. Processing time varies based on document size and complexity.

### 3. Review Results

Results display in a split view:
- **Left:** Raw Markdown source
- **Right:** Rendered preview
- **Bottom:** Extracted images (if any)

### 4. Download

Choose your download format:
- **Download:** Single Markdown file (or selected file in batch mode)
- **Download ZIP:** All files as archive (batch mode)

## Tips for Best Results

- Use high-quality, clear documents
- Ensure text is readable (not blurry or rotated)
- For multi-page PDFs, all pages are processed
- Complex formatting may not translate perfectly to Markdown
